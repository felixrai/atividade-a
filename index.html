1. introdução a sistemas operacionais
    1. O que é ?
      • é um programa que gerencia o hardware do computador
      • fornece a base para o desenvolvimento dos programas de aplicação
    2. Qual sua importância ?
      • proteger a máquina do usuário
      • assume o controle do PC analisando os dispositivos intalados
    3. Tipos de sistemas operacionais
      • sistemas monoprogramáveis
      • multiprogramáveise e múltiplos processadores


  2. Tipos de sistemas operacionais
    1. Sistemas monoprogramáveis/monotarefa
      • recursos do sistema ligado apenas uma tarefa
      • MS-DOS
    2. Sistemas multiprogramáveis/multitarefa
      • mais de um usúario pode interagir
      • Linux, Windows
    3. Sistemas com múltiplos processadores
      • vários programas ao mesmo tempo
      • programa dividido em partes


  3. Funções básicas
    1. Interpretar os comandos
      • principal ligação entre o usuário
      • executa comandos lidos do dispositivo de entrada
    2. Controlar os periféricos
      • recebem ou enviam informações para o computador
      • impressoras, leitores de CD, mouses etc
    3. Organiza arquivos em disco
      • armazenamento de dados com acesso aos dados como uma corrente
      • mapear arquivos em disco, fazer a manipulação de arquivos e diretórios


  4. hardware e software
    1. O que são ?
      • hardware e software são elementos vitais em um sistema de computação
      • um não vive sem o outro
    2. hardware
      • compõem um computador
      • monitor, teclado, processador etc
    3. software
      • executa tarefas específicas
      • calculadora, chrome, microsoft


  5. Sistemas operacionais usuais
    1. Microsoft Windows
      • facilidade no uso, sistema de permissão
      • suporta vários tipos de hardware
    2. Mac OS X
      • sistema para computadores macintosh
      • gerenciar arquivos, executar seus app etc
    3. Linux
      • flexibilidade, seguro etc
      • sistema operacional de código aberto e gratuito
